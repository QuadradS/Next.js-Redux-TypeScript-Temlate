# SSR boilerplate with Next.js

A boilerplate of Server Side Rendering with Next.js

# Libraries

- now
- React
- Redux
- Next
- emotion
- Jest
- TypeScript

# Setup

```
git clone git https://github.com/QuadradS/Next.js-Redux-TypeScript-Temlate.git
cd next-redux-typescript-starter
yarn
```

# Development

```
yarn dev
```

# Production

```
yarn deploy
```
